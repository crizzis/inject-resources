Artifacts belonging to `@InjectResources` are hosted on _JCenter_ so you need to set it up as a repository in your
_Maven/Gradle_ project.

For Maven:

[source, xml]
----
<repositories>
    <repository>
        <snapshots>
            <enabled>false</enabled>
        </snapshots>
        <id>jcenter</id>
        <name>bintray</name>
        <url>https://jcenter.bintray.com</url>
    </repository>
</repositories>
----

For Gradle:

[source, groovy]
----
repositories {
    jcenter()
}
----
